# NOOSE-Website · Funktionsübersicht

Kompakte Liste aller geplanten Funktionen (Details & Phasen siehe [`Plan.md`](Plan.md)).

### Accounts & Rechte
- **Discord-Login** - Anmeldung über Discord
- **Registrierung + Freigabe** - Admin schaltet frei
- **Dienstgrad-Rechte** - Rechte je Rang
- **Nutzerverwaltung** - Accounts/Ränge verwalten
- **Notfall-Sperre** - Account sofort sperren
- **Discord-Rollen-Sync** - Rang ↔ Discord-Rolle
- **Verschlusssachen** - geheime Akten
- **Personalakte** - Dienstgrad-Verlauf je Agent
- **Beförderungs-Workflow** - beantragen/genehmigen
- **TRU-Flag** - rangübergreifende Einheit

### Personen-Akten
- **Personenakte** - zentrale Akte je Person
- **Erweiterter Steckbrief** - Aliase, Telefon, Fahrzeuge, Waffen
- **Foto-Galerie** - mehrere Bilder
- **Personen-Doks** - Verhör-/Maßnahmen-Protokolle
- **Einstufung + Verlauf** - Prüf-/Verdachts-/staatsgefährdend
- **Maßnahmen-Ausgang** - Spritze/offiziell/erschossen/laufen
- **Personen-Beziehungen** - Familie/Feind/Verbündeter
- **Duplikat-Erkennung** - Warnung bei Doppeln
- **Akten zusammenführen** - Merge
- **Quick-Add** - Blitz-Erfassung

### Gruppen, Fraktionen, Parteien
- **Personengruppen** - mit Erfassungsfortschritt
- **Fraktionen** - Bestände, Ränge, Konflikte
- **Erkennungsfarbe** - Farbmarkierung
- **Bedrohungs-Score** - automatischer Gefährdungswert
- **Parteien** - politische Parteien
- **Konflikte** - verknüpfte Gegnerschaften

### Fälle & Einsätze
- **Fall-/Vorgangsakten** - bündelt mehrere Akten
- **Operationen** - Einsatzberichte
- **Observationsprotokoll** - Überwachungseinträge
- **Taskforces** - Einheiten mit Genehmigung
- **Antrags-Workflow** - Posteingang für Freigaben

### Verknüpfung & Suche
- **Interne Verlinkung** - alles bidirektional
- **Quellen/Anhänge** - Upload/Link/intern/Text
- **Volltextsuche** - über alle Inhalte
- **Tags/Labels** - Schlagwörter
- **Gespeicherte Suchen** - Smart-Listen
- **Command-Palette** - Schnellsprung (Strg+K)
- **Verknüpfungs-Vorschläge** - zusammenhängende Akten
- **Pfad-Suche** - „wie hängen A & B zusammen?"

### Übersicht & Auswertung
- **Dashboard** - Kennzahlen + Schnellsuche
- **Aktivitäts-Feed** - „Was ist neu"
- **Statistiken/Charts** - Auswertungen
- **Reports/Export** - CSV/PDF
- **Automatischer Lagebericht** - Monatsbericht
- **Beziehungs-Graph** - grafisches Netzwerk
- **Organigramm** - NOOSE-Struktur
- **Zeitstrahl** - Ereignisse chronologisch
- **Karte mit Orten** - Marker auf GTA-Karte
- **Kalender/Termine** - Termine planen

### Zusammenarbeit
- **Kommentare/Vermerke** - Notizen an Akten
- **@-Erwähnungen** - Kollegen benachrichtigen
- **Aufgaben/To-Dos** - Aufgaben zuweisen
- **Watchlist** - Akten folgen
- **Benachrichtigungen** - Glocke im System
- **News/Schwarzes Brett** - Ankündigungen
- **Behörden-Broadcast** - Rundnachricht

### Aktualität & Nachvollziehbarkeit
- **Änderungs-Historie** - wer/was/wann
- **„Zuletzt aktualisiert"** - Zeitstempel
- **Zugriffsprotokoll** - wer hat angesehen
- **Aktualitäts-Ampel** - veraltet + Wiedervorlage
- **Papierkorb** - Soft-Delete, wiederherstellbar

### Wissen & Komfort
- **Vorlagen/Templates** - einheitliche Masken
- **Custom-Felder** - eigene Felder ohne Code
- **Datei-Bibliothek** - zentrale Ablage
- **Gesetzbuch** - Rechtsgrundlagen, verknüpfbar
- **PDF-Export** - Akte als PDF

### Admin & System
- **Basisdaten-Verwaltung** - Listen pflegen
- **Theming/Logo-Upload** - Look ohne Code
- **Wartungsmodus + Banner** - Seite sperren/Hinweis
- **Partner-Lesezugriff** - DoJ/LSPD/LSMD nur lesen
- **Deployment/Backups** - VPS, HTTPS, Sicherung
